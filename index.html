The Problem
Most people overspend without realising it because they have no structured system to track daily expenses. Generic spreadsheets feel complex, and most budgeting apps are either too heavy or inaccessible. Small daily purchases add up unnoticed, savings goals remain vague, and financial decisions are reactive instead of planned.
✦
The Solution
Bachat Buddy is a focused, lightweight web application that lets users manually log income and expenses, instantly see their balance, analyse spending through visual graphs, set savings goals, receive budget alerts, and get AI-based financial suggestions — all in one clean, easy-to-use interface without complexity or jargon.
Key Features
💰
Budget Tracking

  • Manually add expense entries with category tags
  • Category-based classification: Food, Transport, Utilities, etc.
  • Real-time balance calculated after every single entry
📊
Visual Graph Analysis
  • Income vs Expense bar chart updated in real time
  • Monthly spending breakdown by category
  • Clear financial summary with totals at a glance
  • Trend view to spot overspending periods instantly
🔔
Smart Notifications
  • Alert when spending nears or exceeds set budget limit
  • Unusual spending pattern notifications triggered automatically
  • Reminders to log expenses and stay consistent
  • Warnings when savings pace falls behind goal timeline
🎯

  • Set personal savings goals — phone, trip, emergency fund, etc.
  • Visual progress bar showing exact percentage completion
  • Estimated goal completion date based on current savings rate
  • Track multiple goals simultaneously with individual progress

Technical Implementation

How It Was
Built.

Bachat Buddy was developed using core web technologies — HTML, CSS, and JavaScript — with a strong focus on structured logic, validated data entry, clean UI, and a rule-based suggestion engine.

HTML5CSS3JavaScript DOM ManipulationLocal Storage Chart RenderingRule-Based AI Logic
01
Income & Expense Calculation Logic
Each entry is captured through validated form inputs, stored in structured arrays, and used to dynamically recalculate the running balance. Entries are tagged by category at input for accurate reporting downstream.
02
Graph Generation
Spending data is aggregated by month and category, then rendered as visual bar charts using JavaScript DOM manipulation. Charts update live as new entries are added by the user.
03
Goal Tracking Logic
Users set a target amount and deadline. The system calculates required monthly savings rate, tracks current progress, updates the progress bar, and estimates the goal completion date dynamically.
04
AI Adviser — Rule-Based Engine
A lightweight rule-based logic engine analyses income-to-expense ratio and per-category totals. When any category exceeds a defined threshold, a contextual tip is surfaced — for example: "Your transport costs are high this month. Consider carpooling or public transit to save ₹1,500+."
